Investment Scores

Another side project I created and still maintain at my current position is a scoring system for ETFs and mutual funds. When my colleagues researched funds, I saw them look at several data points for each fund and try to combine all of these distinct values and mentally make a picture of how the fund performs overall. The objective of the system I created is to combine those data points - return, standard deviation, yield, expense ratio, etc - into one easily digestible number, centered around zero.

List Structure

Our company primarily uses TD Ameritrade as a custodian, so the only mutual funds used in the list are their commission-free mutual funds. In September of 2019, TD Ameritrade removed commission fees from ETFs and stocks, so all United States ETFs in Morningstar’s database (the data source I use) are used. As of 11/30/2019, there are 6,239 such mutual funds and 2,391 such ETFs overall.

Scoring

For each of the data points used, an individual score is calculated1, with the average set to be zero. These individual scores are then summed to return to total score of the fund. Because each element of the score is centered around zero, the final score is as well.

1. Many of the data points used match other data points, but with a different trailing period (e.g. 15-year annualized return and 1-year return are both used). For these elements, only one score comprising each trailing period is obtained instead of a separate score for each trailing period. The calculations for these scores use each trailing period, but are designed to place a higher emphasis on the period with the longer history. As an example, the total Return score places the heaviest weight on 15-year annualized return, less of a weight on 10-year annualized return, even less of a weight on 5-year annualized return, etc. This helps get a fuller picture of the investment while not completely ruling out its recent performance.
